윈도우에서 DNS 변경하는 방법

  1. 네트워크 설정 열기

    작업표시줄 우측 하단 인터넷 아이콘 우클릭 → "네트워크 및 인터넷 설정 열기" 클릭

  2. 어댑터 설정 변경

    "어댑터 옵션 변경" → 사용 중인 네트워크(이더넷 또는 Wi-Fi) 우클릭 → "속성" 클릭

  3. 인터넷 프로토콜 버전 4 선택

    "인터넷 프로토콜 버전 4(TCP/IPv4)" 선택 후 "속성" 클릭

  4. DNS 주소 입력

    "다음 DNS 서버 주소 사용" 선택 → 기본 DNS에 8.8.8.8, 보조 DNS에 8.8.4.4 입력 → 확인

  5. 브라우저에서 야코레드 접속 시도

    브라우저를 재시작하고 야코레드에 접속해 보세요.

방법 2. VPN 사용

VPN은 모든 차단을 우회할 수 있는 가장 확실한 방법입니다

🛡️
무료

ProtonVPN

프라이버시를 중시하는 스위스 기반 VPN. 무료 플랜 제공. 로그 없음 정책.

protonvpn.com에서 무료 다운로드

🔐
무료

Windscribe VPN

월 10GB 무료 제공. 설치 없이 크롬 확장 프로그램으로도 사용 가능.

windscribe.com에서 무료 다운로드

🌏
무료

Psiphon

검열 우회에 특화된 무료 VPN. 앱 다운로드 없이 실행 파일로 사용 가능.

psiphon3.com에서 무료 다운로드

방법 3. 브라우저 확장 프로그램

설치가 간편하고 빠른 접속이 가능한 브라우저 확장 프로그램 방법입니다

  1. 크롬 웹 스토어 접속

    구글 크롬 브라우저를 열고 크롬 웹 스토어(chrome.google.com/webstore)에 접속합니다.

  2. "VPN" 또는 "Proxy" 검색

    검색창에 "VPN" 또는 "free proxy"를 검색하고 평점 높은 확장 프로그램을 선택합니다.

  3. 확장 프로그램 설치

    "Chrome에 추가" 버튼을 클릭하여 확장 프로그램을 설치합니다.

  4. VPN 연결 후 접속

    확장 프로그램을 활성화하고 야코레드에 접속하세요.

💡 추천 확장 프로그램: Browsec VPN, Hola VPN, SetupVPN 등이 무료로 이용할 수 있으며 설치가 간편합니다.
⚠️ 주의: 무료 VPN은 속도 제한이 있을 수 있습니다. 중요한 개인정보 입력 시 유료 VPN 사용을 권장합니다.

자주 묻는 질문

DNS 변경이 어렵습니다. 더 쉬운 방법이 있나요?
브라우저 확장 프로그램 방법이 가장 쉽습니다. 크롬 웹 스토어에서 "Browsec" 또는 "Hola VPN"을 검색하여 클릭 한 번으로 설치할 수 있습니다.
스마트폰에서 야코레드 우회 접속하는 방법은?
스마트폰에서는 VPN 앱을 다운로드하는 것이 가장 편합니다. 안드로이드는 ProtonVPN, 아이폰은 App Store에서 VPN 앱을 검색하여 설치하세요.
DNS 변경 후에도 접속이 안 됩니다.
브라우저 캐시를 지우거나(Ctrl+Shift+Delete), 브라우저를 재시작해 보세요. 그래도 안 되면 VPN 방법을 시도해 보세요.
VPN 사용이 불법인가요?
한국에서 VPN 사용 자체는 불법이 아닙니다. 개인 프라이버시 보호와 차단된 콘텐츠 접속을 위한 VPN 이용은 합법입니다.
